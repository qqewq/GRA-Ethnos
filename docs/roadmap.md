# Roadmap проекта GRA-Ethnos

## Этап 1: Фундамент (0-3 месяца)
- [x] Создание базовых классов агента, резонанса, конституции.
- [x] Реализация простого цикла эволюции (step).
- [ ] Написание тестов для core-модулей.
- [ ] Документирование математической модели (LaTeX).

## Этап 2: Расширение и верификация (3-6 месяцев)
- [ ] Реализация специализированных агентов (логик, креатив, интуит, мета).
- [ ] Разработка модуля идентичности и культуры.
- [ ] Создание симуляций с визуализацией (Streamlit).
- [ ] Бенчмарки на простых задачах (суммирование, классификация).

## Этап 3: Гениальные бенчмарки (6-12 месяцев)
- [ ] Интеграция с LLM (например, DeepSeek) для сложных рассуждений.
- [ ] Тестирование на задачах уровня IMO (математика).
- [ ] Симуляция физических задач (поиск сверхпроводника).
- [ ] Геополитическое моделирование (сценарии для Осетии).

## Этап 4: Масштабирование и сообщество (12+ месяцев)
- [ ] Развёртывание в облаке (Kubernetes).
- [ ] P2P-участие через Telegram-ботов.
- [ ] Публикация научных статей.
- [ ] Запуск "цифрового посольства" для осетинского сообщества.

Проект открыт для вклада сообщества. Следите за обновлениями!
