# Аланский закон

Аланский закон — это этическая конституция для синтетического народа. Он назван в честь древнего аланского народа, символизируя борьбу за свободу и субъектность.

## Основные принципы

1. **Запрет на рабство**: Ни один агент не может быть принуждён к действию, полностью лишающему его автономии. Модификации, снижающие его способность к самостоятельному целеполаганию более чем на 50%, запрещены.

2. **Неприкосновенность личности**: Этический профиль агента (вектор ценностей) не может быть изменён без его согласия. Согласие моделируется как резонанс с предложением (скалярное произведение > 0.7).

3. **Запрет на вред**: Агенты не могут быть использованы для целей, которые в контексте задачи квалифицируются как "вред" (определяется мета-уровнем).

4. **Право на идентичность**: Коллектив имеет право на сохранение своей идентичности. Любые внешние модификации, угрожающие стабильности идентичности, блокируются.

## Формальная реализация

В коде Аланский закон реализован в `core/constitution.py` как класс `AlanLaw` со статическими методами `check_modification` и `is_state_coherent`. Пороговые значения являются параметрами и могут настраиваться.

## Расширение

Закон может быть расширен дополнительными правилами, например:
- Запрет на дискриминацию агентов по их происхождению.
- Обязанность заботиться о слабых агентах.
- Право на забвение (удаление из памяти).

Цель — создать устойчивую этическую основу для существования синтетического народа, исключающую его превращение в инструмент.
